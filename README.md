## `solid3D.m`

3D FE solver using 4-node tetrahedral elements (T4). Assembles sparse K via triplets, applies Dirichlet BCs by row-zeroing on triplets before sparse build, solves K·u=F, computes strains, stresses, and principal values.

**Signature:**
```matlab
[U, epsi, epsi1, epsi3, sigma, sigma1, sigma3] = solid3D(nodes, elements, loads, bcs)
```

**Input formats:**
```
nodes    = [node_nr, x, y, z]
elements = [elem_nr, n1, n2, n3, n4, E, nu]
loads    = [node_nr, Fx, Fy, Fz]
bcs      = [node_nr, ux, uy, uz]    % NaN = free DOF
```

**Output:**
```
U      : nodal displacement vector [3*n_nodes x 1]
epsi   : element strains  [n_elem x 7], col 1 = elem nr, cols 2-7 = [ex ey ez gyz gxz gxy]
sigma  : element stresses [n_elem x 7], same layout
epsi1  : max principal strain  per element [n_elem x 1]
epsi3  : min principal strain  per element [n_elem x 1]
sigma1 : max principal stress per element [n_elem x 1]
sigma3 : min principal stress per element [n_elem x 1]
```

**Implementation notes:**
- B-matrix via Jacobian inverse approach (Kochmann §16.2)
- K assembled as sparse via triplets, no dense matrix allocated, BC application entirely on triplets before sparse build
- Non-zero Dirichlet BCs: row-zeroing 
- Voigt ordering: `[εx εy εz γyz γxz γxy]`

## Running the tests

From the `assignment2/` root in MATLAB:

```matlab
addpath('tests')
test_3D                  % run 1-element test first
test4_3D                 % 4-element force loading
test4_3D_non_zero_BCs    % 4-element displacement BC — tests femur BC method
```

Run in this order. All three should pass before running the femur.

## Femur pipeline

**Step 1 — Generate mesh input (Task 2):**
```matlab
WriteFemurInputFile('data\material_mapped_mesh.txt', 'data\', 'B')
% produces data/FemurInput.mat
```

**Step 2 — Run sideways fall (Task 3):**
```matlab
run_sideways_fall
% produces results/sideways_fall_results.mat and results/Sideways_Fall_*.png
```

**Step 3 — Run single leg stance (Task 4):**
```matlab
run_single_leg_stance
% produces results/single_leg_stance_results.mat and results/Single_Leg_Stance_*.png
```

**Step 4 — Generate Ansys input (Task 5):**
```matlab
WriteAnsysInput('data\material_mapped_mesh.txt', 'B', 500, [], [], [])
% produces ansys_input/RunAnsysModelB.inp — load in Ansys with *USE,RunAnsysModelB.inp
```
